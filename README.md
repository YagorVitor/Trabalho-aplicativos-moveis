# Backlog de Jogos

Mini app em React Native/Expo para organizar jogos em três status:
**Quero jogar**, **Jogando** e **Zerado**.

## Como executar

```bash
npm install
npx expo start
```

## Requisitos do trabalho atendidos

- `FlatList` com `keyExtractor`
- `TextInput` controlado com `useState`
- Botão para adicionar itens
- Layout com Flexbox separando entrada e lista
- Estilização com `StyleSheet.create()`

## Bônus implementados

- Botão para remover item
- Status visual por categoria com badge
- Mensagem amigável quando a lista está vazia
- Filtros por status
- Persistência local com `AsyncStorage`
- Interface dark moderna com gradiente e cards

## Tema

O app foi pensado como um backlog pessoal de jogos, com visual escuro e mais polido para melhorar a apresentação no envio.
